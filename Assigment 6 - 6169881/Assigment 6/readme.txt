All information is found at the program interface.
Please when opening the file follow the instructions of the menu.

Make sure to always enter the Absolute path, relative path do not always work.

If a word is found you will see the following message:
---------------------------------------------
RESULT: [INDEX OF THE WORD]
THE WORD [WORD] WAS FOUND AT INDEX [INDEX]
---------------------------------------------


If a word is NOT found you will see the following message:
---------------------------------------------
RESULT: -1
THE WORD [WORD] WAS NOT FOUND 
---------------------------------------------

IMPORTANT: Index refers to the index of the array, returned by the binary search. The actual position of the word is always INDEX+1.
